# Markdown 语法总结

本科用了四年md都没有总结过它的技巧，趁现在有空简单记录一下。

## 一、标题

一级标题：#+空格+标题

二级标题：##+空格+标题

三级标题：###+空格+标题

最多支持六级标题。

## 二、字体

粗体：文字两边各加两个*或ctrl+b

斜体：文字两边各加三个*或ctrl+i

删除线：文字两边加两个~

下划线：ctrl+u

## 三、引用

箭头：>+加引用内容

>test

## 四、分割线

三个杠：---

---

## 五、图片

![图片名字]+(图片地址)

## 六、超链接

 [点击跳转到xx]+(网址)

## 七、列表

有序列表：.+空格

1. apple
2. banana

无序列表：-+空格

- orange

## 八、表格

|      |      |      |
| ---- | ---- | ---- |
|      |      |      |
|      |      |      |
|      |      |      |

typora右键插入

## 九、代码

代码块：```+java

```java
public
```
